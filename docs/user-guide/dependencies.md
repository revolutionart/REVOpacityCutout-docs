
## Dependencies Status In The Panel

REVO Opacity Cutout already includes its required Python wheels:

- OpenCV (`opencv-python`)
- PyClipper (`pyclipper`)
- numpy

The **Dependencies Status** section in the addon UI is informational only.

It shows whether Blender can currently import the bundled packages.
